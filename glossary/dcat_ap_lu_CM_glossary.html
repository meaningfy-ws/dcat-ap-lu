Class name Definition

prov:Attribution Attribution is the ascribing of an entity to an agent.
skos:Concept An idea or notion; a unit of thought.
spdx:Checksum A Checksum is value that allows the contents of a file to be authenticated. Even small changes to the content of the file will change its checksum. This class allows the results of a variety of checksum and cryptographic message digest algorithms to be represented.
vcard:Kind A description following the vCard specification, e.g. to provide telephone number and e-mail address for a contact point.
